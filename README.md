# 📚 StudyNet

> 함께 공부하고 성장하는 **스터디 네트워크 모바일 앱**

StudyNet은 개인의 공부 시간을 기록하고, 친구들과 함께 공부하며 랭킹 시스템을 통해 학습 동기를 높일 수 있는 **모바일 스터디 관리 앱**입니다.
React Native와 Expo 기반으로 개발된 **생산성 + 커뮤니티형 학습 애플리케이션**입니다.

---

# 🚀 프로젝트 소개

많은 공부 앱들은 단순한 타이머 기능만 제공하지만,
StudyNet은 **학습 기록 + 커뮤니티 + 경쟁 요소**를 결합하여
사용자가 지속적으로 공부할 수 있는 환경을 만드는 것을 목표로 합니다.

핵심 컨셉

* 📚 함께 공부하는 **스터디룸**
* 👥 친구와 공부 기록 공유
* 🏆 랭킹을 통한 학습 동기 부여
* 📊 개인 공부 기록 관리

---

# ✨ 주요 기능

### ⏱ 공부 시간 기록

* 공부 시간을 기록하고 저장
* 하루 공부 시간 확인
* 연속 공부일(Streak) 관리

### 📚 스터디룸

* 여러 사용자가 함께 공부할 수 있는 공간
* 공동 학습을 통한 집중 환경 제공

### 👥 친구 기능

* 친구 추가
* 친구의 공부 활동 확인

### 🏆 랭킹 시스템

* 공부 시간 기반 사용자 랭킹
* 경쟁을 통한 학습 동기 강화

### 📊 학습 기록

* 공부 시간 통계 확인
* 개인 학습 데이터 관리

# 🔧 실행 방법

```bash
git clone https://github.com/hun0493/Study_Net.git
cd Study_Net
npm install
npx expo start
```

Expo 실행 후

* Android Emulator 실행
* 또는 실제 스마트폰에서 Expo Go 앱으로 실행
